--- layout: post name: mylyn-google-code title: mylyn 连接 google code 出错的解决办法 date: 2009-01-17 13:21:00.000000000 +08:00 categories: - google code - eclipse - mylyn permalink: /2009/01/mylyn-google-code.html --- 通过 mylyn 的 Generic Web Connector(从[1]安装) 可以连接 Google Code, 不过最近没法正常连接, 解决方案如下(参考[2]):

修改 repository 的属性, 修改其中的 Additional Settings -> Advanced Configuration 中的 "Query Request URL" 和 "Query Pattern" 两项, 分别修改为
  • Query Request URL: ${serverUrl}/csv?colspec=ID+Status+Type+Owner+Summary
  • Query Pattern: "({Id}[0-9]+?)","({Status}.*?)","({Type}.*?)","({Owner}.*?)","({Description}.*?)"
